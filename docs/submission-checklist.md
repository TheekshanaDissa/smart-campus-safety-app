# Submission Checklist

## Source code

- [ ] Zip source code without `node_modules`
- [ ] Include mobile app source
- [ ] Include backend source
- [ ] Include Firebase rules
- [ ] Include `.env.example`
- [ ] Do not include `.env`
- [ ] Do not include service account keys

## GitHub evidence

- [ ] Repository link
- [ ] Sprint branches or commits
- [ ] Screenshots of individual contributions

## Azure DevOps or similar

- [ ] Sprint boards
- [ ] User stories
- [ ] Testing tasks

## User manual

- [ ] Getting started instructions
- [ ] App usage instructions
- [ ] Troubleshooting table

## Testing and deployment report

- [ ] Unit testing evidence
- [ ] Integration/API testing evidence
- [ ] End-to-end testing evidence
- [ ] Firebase Test Lab evidence
- [ ] Reflection on issues and improvements

## Presentation

- [ ] 5-minute pitch slides
- [ ] Demonstrate app instead of only showing screenshots
- [ ] Include testing and results
- [ ] Include lessons learned and future improvements

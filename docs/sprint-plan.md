# Agile Sprint Plan

## Project: Smart Campus Safety App

## Sprint 1: Weeks 3–5 — Planning, UI, Navigation

### Goal
Create the basic app structure, user flows, and navigation.

### User stories

| ID | User story | Priority | Status |
|---|---|---:|---|
| US01 | As a student, I want to register/login so that I can submit reports securely. | High | Done |
| US02 | As a student, I want to navigate between home, report, map and history screens. | High | Done |
| US03 | As a student, I want a clean dashboard showing safety report summary. | Medium | Done |

### Tasks

- Create Expo React Native app.
- Add React Navigation.
- Design login/register screens.
- Build dashboard and tab navigation.
- Prepare GitHub repository and branches.

### Output
Functional navigation and UI screens.

---

## Sprint 2: Weeks 6–8 — Firebase and Device Features

### Goal
Integrate Firebase and mobile device capabilities.

### User stories

| ID | User story | Priority | Status |
|---|---|---:|---|
| US04 | As a student, I want to submit incident reports to Firestore. | High | Done |
| US05 | As a student, I want to attach my GPS location to a report. | High | Done |
| US06 | As a student, I want to capture photo evidence using the camera. | Medium | Done |
| US07 | As a student, I want to view incidents on a map. | High | Done |
| US08 | As a student, I want to receive notification feedback after submission. | Medium | Done |

### Tasks

- Configure Firebase Auth.
- Configure Firestore.
- Add GPS location service.
- Add camera capture.
- Add map markers.
- Add local notifications.

### Output
Main app features completed.

---

## Sprint 3: Weeks 9–11 — Backend, Testing, Documentation, Deployment

### Goal
Add backend API, tests, final polish and documentation.

### User stories

| ID | User story | Priority | Status |
|---|---|---:|---|
| US09 | As an admin/backend service, I want an API to read and update incident status. | Medium | Done |
| US10 | As a developer, I want tests to verify validation and backend routes. | High | Done |
| US11 | As a marker, I want clear documentation and setup instructions. | High | Done |

### Tasks

- Build Node.js Express API.
- Add Firebase Admin SDK.
- Add backend tests with Jest and Supertest.
- Add mobile tests with Jest.
- Prepare user manual, testing report and pitch outline.

### Output
Submission-ready app with source code and documentation.
